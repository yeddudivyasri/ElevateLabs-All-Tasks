# Notes App – Java File I/O

## Objective

A text-based Notes Manager developed in Java that demonstrates File I/O operations using FileReader, FileWriter, BufferedReader, and BufferedWriter.

## Features

* Add new notes
* View all saved notes
* Search notes by keyword
* Delete a specific note
* Clear all notes
* Automatic timestamp for each note
* Notes stored in a text file

## Technologies Used

* Java
* FileReader
* FileWriter
* BufferedReader
* BufferedWriter
* VS Code
* Terminal

## Concepts Covered

* File Handling
* Exception Handling
* Collections (ArrayList)
* Loops and Conditions
* CLI-based Application

## How to Run

Compile:
javac NotesApp.java

Run:
java NotesApp

## Output

The application stores all notes in a file named `notes.txt` and allows users to manage notes through a command-line interface.
